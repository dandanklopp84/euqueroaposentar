# euqueroaposentar — Conferidor Mega 2025 (versão focada)

Apenas conferência dos jogos: 6 campos de entrada, destaque visual automático dos acertos por jogo e cálculo de quadras/quinas/senas. Ordena por: mais acertos → maior tamanho do jogo.

Publicar no GitHub Pages: Settings → Pages → Deploy from a branch → main → /(root).
